# <a name="focused-inbox-in-outlook"></a>重点收件箱Outlook

重点收件箱将收件箱分为两个选项卡：重点选项卡和其他选项卡。 最重要的消息位于"重点"选项卡上，其余邮件仍易于访问（但无法访问）在"其他"选项卡上。
