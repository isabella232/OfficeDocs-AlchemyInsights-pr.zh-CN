# <a name="check-for-app-using-camera"></a>使用相机检查应用

1. 选择 **"开始**"，然后选择 **"设置 >隐私>相机** (ms-settings：privacy-webcam) 
2. 在应用程序列表中，查找"当前使用"的应用，然后关闭该应用程序。
