# <a name="change-your-profile-name-in-outlookcom"></a>在 Outlook.com 中更改配置文件名称

1. 转到 [你的 Microsoft 帐户信息](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 选择 **"编辑名称"。**
3. 输入名字和姓氏，**然后选择保存。**

请允许最多 24 小时对配置文件名称的更改生效。
