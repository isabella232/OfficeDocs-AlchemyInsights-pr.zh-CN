# <a name="configure-privilege-for-user-and-system-chart"></a>为用户和系统图表配置特权

**症状**：没有图表的读取权限。

**原因**：特权不足或不包含必要的特权错误。

**解决方案**：配置图表的特权

**步骤**：

1. 转到 **设置 -> 自定义 -> 自定义系统 -> 安全角色**。

2. 双击角色。

    - 用户图表特权配置位于“**核心记录**”选项卡下方。
    - 系统图表特权配置位于“**自定义**”选项卡下方。
