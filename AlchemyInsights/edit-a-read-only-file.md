# <a name="edit-a-read-only-file"></a>编辑只读文件

在文档库中，当您单击文档的名称时，Office程序可能会意外地以只读文件打开它。 若要打开它进行编辑，请单击"打开 (三个点) ，然后单击"编辑 **"。**
