# <a name="yammer---possible-service-issue"></a>Yammer - 可能的服务问题

可能有一个临时问题，即用户在访问网站时可能会遇到Yammer。 检查 [服务运行状况仪表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) ，了解这是否会影响网络。
