# <a name="microsoft-365-for-nonprofits---declined"></a>Microsoft 365非营利组织版 - 拒绝

我们审查你的请求，确定此组织不符合成为 Microsoft 合格非营利组织的资格。
  
若要以Office 365购买产品，请参阅[可用的产品/服务](https://portal.office.com/AdminPortal/Home)。
