# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>已从 Outlook 网页版 (OWA) 中删除公用文件夹

每次注销或屏幕刷新时，OWA 中的公共文件夹都会从收藏夹中删除，这是一个已知的问题。 该问题将很快得到修复。 在此之前，请重新添加该文件夹以从 OWA 访问。
