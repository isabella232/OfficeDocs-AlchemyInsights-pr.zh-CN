# <a name="create-users-in-hybrid-deployments"></a>在混合部署中创建用户

如果您的组织是本地环境和 Microsoft 365之间的混合部署，则需要在本地 Active Directory 中创建所有用户对象。 在组织单位或 OU 中 (用户) ，以便它们同步到组织中。
