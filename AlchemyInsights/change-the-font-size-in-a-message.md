# <a name="change-the-font-size-in-a-message"></a>更改邮件中的字号

1. 如果已键入邮件，请选择文本。
2. 选择邮件  **下方的"** 格式"图标 (该图标是邮件的第一个) 。
3. 选择邮件  **上方**  的"字号"图标 (它是第七个) 。
4. 选择大小。
