# <a name="onedrive-recycle-bin"></a>OneDrive 回收站

从 OneDrive 删除的项目或网站不会立即删除。 删除的项目会在回收站保留一段时间。 在此期间，可以将已删除的项目还原到原始位置。 [了解详细信息](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f?ui=en-US&rs=en-US&ad=US)。
