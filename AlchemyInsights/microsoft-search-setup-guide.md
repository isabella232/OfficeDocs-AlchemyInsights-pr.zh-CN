# <a name="microsoft-search-setup-guide"></a>Microsoft 搜索设置指南

使用[Microsoft 搜索设置指南](https://go.microsoft.com/fwlink/?linkid=2153798)来设置和配置Microsoft 搜索。 是希望向一组用户试用还是向组织中的每个人推出它。 你将分配搜索管理员和搜索编辑器，然后使用答案和其他选项为用户自定义搜索体验，例如将 必应 扩展添加到 Chrome 或将 必应 设置为默认搜索引擎。
