# <a name="how-to-select-all-in-outlookcom"></a>如何在 Outlook.com 中"全选"

只有在所有邮件都可见时，才能全选。 这是为了防止对看不到的邮件执行意外操作。

**Ctrl + 单击** 选择多个单独邮件。

**Shift + click** 选择您选择的第一封邮件和最后一封邮件之间的所有邮件。
