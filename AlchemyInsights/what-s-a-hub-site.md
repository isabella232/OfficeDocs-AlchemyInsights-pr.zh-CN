# <a name="whats-a-hub-site"></a>什么是中心网站？

中心网站可帮助你组织网站，并使它们具有常见的导航和品牌。 全局SharePoint管理员将能够创建中心网站并指定哪些用户可以将其网站与中心网站关联。 有关详细信息，请参阅SharePoint[中心网站。](https://go.microsoft.com/fwlink/?linkid=869388)
