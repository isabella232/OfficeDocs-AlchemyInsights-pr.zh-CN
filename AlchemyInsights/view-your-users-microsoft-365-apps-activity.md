# <a name="view-your-users-microsoft-365-apps-activity"></a>查看用户的 Microsoft 365 应用活动

[Office 激活报表](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide)将显示在至少一个设备上安装并激活了其 Microsoft 365 应用的用户。

此外，你可以使用 [Microsoft 365 应用使用情况报表](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)查看用户的 Microsoft 365 应用活动。
