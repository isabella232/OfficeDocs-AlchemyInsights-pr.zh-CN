# <a name="stream-quality"></a>Stream 质量

Stream 利用 HTTPS 单播自适应比特率流式播放 (ABR)，基于可用网络带宽和视频播放器大小动态选择最佳的视频播放质量。 有关详细信息，请参阅：

- [Microsoft Stream 视频传递和网络概述](https://docs.microsoft.com/stream/network-overview)
