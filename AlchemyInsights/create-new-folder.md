# <a name="create-a-new-folder-in-outlookcom"></a>在 Outlook.com 中创建新文件夹

1. 在 **导航窗格中** 的"文件夹"下，选择"**新建文件夹"。**

2. 键入文件夹的名称，然后按 Enter。

若要了解更多信息，请参阅[在 Outlook.com 中创建文件夹](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy)。
