# <a name="remove-an-extension-from-microsoft-edge"></a>从应用程序中删除Microsoft Edge

若要从应用程序中删除扩展Microsoft Edge：

1. 打开 Microsoft Edge。
2. Select **... > Extensions**.
3. 在 **"已安装的扩展"** 页上，找到要删除的扩展，然后选择它旁边的" **删除** "。
4. 在确认对话框中， **选择删除**。
