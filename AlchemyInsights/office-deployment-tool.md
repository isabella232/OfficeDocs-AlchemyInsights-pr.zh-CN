# <a name="deploy-office-to-your-users"></a>将Office部署到用户

使用 Office 部署工具定义部署选项并保存配置文件，以便与对组织具有只读访问权限的用户共享 。 有关详细信息，请参阅 Office[部署工具](https://admin.microsoft.com/AdminPortal/Home#/modernonboarding/cdnwizard)。
