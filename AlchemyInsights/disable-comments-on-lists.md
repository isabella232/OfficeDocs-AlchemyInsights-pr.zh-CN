# <a name="disable-comments-on-lists"></a>禁用列表上的注释

管理员可以在组织级别禁用此功能，方法为更改 **Set-SPOTenant** PowerShell cmdlet 中的 **CommentsOnListItemsDisabled** 参数。

当前不可能在站点或列表级别禁用注释。 我们希望在 2021 年第一季度以后的更新中拥有这些控件。
