# <a name="whats-a-domain"></a>什么是域

域是电子邮件地址的一部分，位于 @ 符号之后。 例如，在地址 rob@contoso.com，域 **contoso.com**。 大多数企业倾向于在电子邮件地址中使用自己的域名。

以下是有关域的一些提示：

- 可以从 Microsoft 购买域名。 这样，你将拥有自定义电子邮件地址。
- 或者，只需将已拥有的域添加到Office 365。
