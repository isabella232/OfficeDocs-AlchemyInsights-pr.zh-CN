# <a name="update-dns-settings"></a>更新 DNS 设置

更新 DNS 设置涉及将Office 365的值复制到域注册机构或 DNS 托管提供商。 为此，请执行下列步骤：

1. 将"更新 **DNS** 设置"页上显示的值复制并粘贴到您的 DNS 托管提供商中。
2. 单击"**验证"。**
3. 单击“完成”。
