# <a name="set-up-online-services"></a>设置联机服务

若要设置联机服务，请执行以下步骤：

1. 在 **"为您的网站添加记录"页上** ：a. 如果没有 **网站**，请单击"下一步"。 然后，您将登录"更新 **DNS 设置"** 页。
    b. 如果您具有网站，则查找 DNS 托管提供商的记录。 然后，将其复制并粘贴到页面中。
2. 单击 **"下** 一步"转到 **"更新 DNS 设置"** 页。
