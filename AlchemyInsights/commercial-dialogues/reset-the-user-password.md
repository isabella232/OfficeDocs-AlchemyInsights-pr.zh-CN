# <a name="reset-the-users-password"></a>重置用户密码

你将需要重置用户密码，然后才能执行任何其他操作。 操作步骤如下：

1. 选择 **"用户**  >  **[""活动用户"。](https://go.microsoft.com/fwlink/p/?linkid=834822)**
1. 选择帐户遭到入侵的用户。
1. In the flyout that appears， click **Reset password**.

> [!NOTE]
> 不要通过电子邮件将密码发送给用户。
