# <a name="my-status-always-shows-as-available"></a>我的状态始终显示为"可用"

每次登录到 Outlook.com 时，帐户状态将重置为"可用"。 可以通过不退出 Outlook.com 来防止状态更改。 若要在关闭浏览器时保持登录状态，请选择Outlook.com 登录页面上的"保持登录状态"框。
