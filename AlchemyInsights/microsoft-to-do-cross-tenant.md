# <a name="microsoft-to-do-cross-tenant-sharing"></a>微软待办跨租户共享

微软待办个人帐户和工作帐户之间的列表共享。 这使贵组织的用户

加入、查看、修改数据以及将数据添加到个人 Microsoft 帐户所拥有的列表中。

管理员可以使用新的管理员设置，通过取消选中该选项来禁用此功能。

管理员可以在管理门户中的"组织""设置"设置"微软待办"。   >    >    >
