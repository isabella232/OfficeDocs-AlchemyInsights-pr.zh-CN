# <a name="delete-a-user-and-remove-licenses"></a>删除用户并删除许可证

删除用户还会取消分配用户许可证，使其可供其他用户使用。 
1. 在管理中心， **转到"活动** >用户"。
1. 选择要删除的一个或多个用户，然后单击"**删除用户"。**

有关详细信息，请参阅[从组织中删除用户](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user)。
