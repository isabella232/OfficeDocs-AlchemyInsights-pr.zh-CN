# <a name="add-in-help"></a>加载项帮助

- 如果特定加载项存在问题，请尝试删除或禁止添加它，然后重新添加。 有关在 Excel 中添加或删除加载项的帮助，请参阅[在 Excel 中添加或删除加载项](https://support.office.com/client/0af570c4-5cf3-4fa9-9b88-403625a0b460)。

- 有关在 Microsoft 365 管理中心部署加载项的帮助，请参阅[在 Microsoft 365 管理中心管理 Office 365 加载项部署](https://docs.microsoft.com/microsoft-365/admin/manage/manage-deployment-of-add-ins)。
