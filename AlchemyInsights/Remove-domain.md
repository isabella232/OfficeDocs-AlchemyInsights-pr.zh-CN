# <a name="trying-to-remove-your-domain"></a>尝试删除你的域？

请按照[删除域](/microsoft-365/admin/get-help-with-domains/remove-a-domain)中的步骤操作。
  
删除域之前，需要更新分配给此域名的所有用户或组。查看[域常见问题](/microsoft-365/admin/setup/domains-faq)，了解其他常见解决方案。
