# <a name="use-microsoft-graph-api-to-manage-groups"></a>使用 Microsoft Graph API 管理组

组是用户和其他主体的集合，他们共享对 Microsoft 服务或应用中资源的访问权限。 Microsoft Graph 提供了可用于根据方案创建和管理不同类型组和组功能的 API。 Microsoft Graph 中所有与组相关的操作都需要征得管理员同意。 有关详细信息，请参阅使用[Microsoft Graph 中的组](https://docs.microsoft.com/graph/api/resources/groups-overview)。
