# <a name="your-users-mailbox-is-being-created"></a>正在创建用户的邮箱

新邮箱可能需要一小时才能准备好使用。
  
如果用户的邮箱已"准备"一个小时以上，可以尝试删除和重新创建用户，或使用"让我们呼叫您"选项与支持工程师交谈。
