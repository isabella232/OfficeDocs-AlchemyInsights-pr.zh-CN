# <a name="what-is-autosave"></a>什么是“自动保存”？

自动保存功能在 Excel、Word 和 Microsoft 365 专属 PowerPoint 订阅者中可用，可在你工作时每隔几秒钟自动保存一次文件。 

有关常见问题，请参阅 [什么是自动保存？](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
