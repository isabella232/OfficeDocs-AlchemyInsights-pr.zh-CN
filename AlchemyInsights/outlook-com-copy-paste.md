# <a name="copy-and-paste-in-outlookcom"></a>复制并粘贴到 Outlook.com 中

Outlook.com 中没有任何复制或粘贴按钮，但可以使用键盘快捷方式：

**在电脑上：** 要复制的 Ctrl+C、要剪切的 Ctrl+X 和要粘贴的 Ctrl+V。

**在 Mac 上：** 要复制的 CMD+C、要剪切的 CMD+X 和要粘贴的 CMD+V。
