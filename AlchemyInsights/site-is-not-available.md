# <a name="site-is-not-available"></a>网站不可用

如果您的SharePoint或OneDrive网站对多个用户不可用，则可能是临时服务问题。 检查 [服务运行状况仪表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) ，查看租户是否受到影响。
