# <a name="delete-a-folder-in-outlookcom"></a>删除 Outlook.com 中的文件夹

1. 在左窗格中，右键单击要删除的文件夹或子文件夹，然后选择"删除 **文件夹"。**
2. 选择 **"确定** "以确认。

若要了解更多信息，请参阅使用[Outlook.com 中的邮件文件夹](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)。
