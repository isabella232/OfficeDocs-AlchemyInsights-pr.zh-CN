# <a name="enable-send-immediately-when-connected"></a>连接后立即启用"发送"
 
1. 在“文件”选项卡上，单击“选项”。

2. 在"Outlook选项"对话框中，单击"**高级"。**

3. 在"发送和接收"部分，单击以启用"**连接时立即发送"。** 单击“**确定**”。
