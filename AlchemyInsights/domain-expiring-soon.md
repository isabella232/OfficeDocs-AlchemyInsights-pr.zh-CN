# <a name="domain-registration-expiration"></a>域注册过期

域过期： 

- 若要验证域过期日期，请咨询域注册机构。

- 如果从 Microsoft 购买域，请转到 [账单>你的产品](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/subscriptions) 以检查域订阅。

- 或者，Whois 查询返回你的域注册到期日期。
