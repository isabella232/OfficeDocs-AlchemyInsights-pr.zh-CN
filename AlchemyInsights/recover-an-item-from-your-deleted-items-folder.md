# <a name="recover-an-item-from-your-deleted-items-folder"></a>从"已删除邮件"文件夹中恢复项目

当您删除电子邮件、联系人、日历项目或任务时，该邮件将移动到"已删除邮件"文件夹。

1. 在 Web 浏览器中使用组织电子邮件管理者提供的 URL 登录到 Outlook Web App。 输入用户名和密码，然后选择"**登录"。**
1. 在电子邮件文件夹列表中，选择 **"已删除邮件"。**
1. 若要还原电子邮件，请执行下列操作之一：a. 若要将邮件还原到收件箱，请右键单击该邮件，然后选择"移动>**收件箱"。**
    b. 若要将邮件还原到其他文件夹，请右键单击该邮件，然后选择"移动>**更多"。** 选择文件夹位置，然后单击"移动 **"。**
4. 还原其他类型的项目：
    - 若要还原联系人，请右键单击该联系人，然后选择"**移动到联系人"。**
    - 若要还原日历项目，请右键单击它，然后选择"**移动到日历"。**
    - 若要还原任务，请右键单击它，然后选择"**移动到任务"。**

**使用技巧"已删除邮件"文件夹中的项目的恢复权限**

- 如果删除某个文件夹，该文件夹将移动到"已删除邮件"文件夹，并显示为子文件夹。 若要在 (及其包含的所有项目) ，请右键单击已删除的文件夹，选择"移动"，然后选择要移动到的文件夹。 
- 管理员可能已经设置策略，在特定天数后从"已删除邮件"文件夹中删除项目。 与删除项目时一样，策略删除的项目将移动到"可恢复的项目"文件夹。 因此，如果在"已删除邮件"文件夹中找不到内容，可在"可恢复的项目"文件夹中查找。
