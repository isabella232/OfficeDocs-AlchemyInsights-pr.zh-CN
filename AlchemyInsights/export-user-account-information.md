# <a name="export-user-account-information"></a>导出用户帐户信息

若要导出 Microsoft 365 管理中心 中的用户帐户信息，包括有关许可证、密码策略、阻止的帐户、AAD Sync 状态等的信息，请转到"用户  >  [""活动](https://go.microsoft.com/fwlink/p/?linkid=834822)用户"页，然后选择"导出 **用户"。**
