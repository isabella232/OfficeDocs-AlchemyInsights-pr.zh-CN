# <a name="search-experience-updated"></a>已更新搜索体验

搜索现在位于应用底部。 通过点击它，你可以直接访问你的顶级联系人、即将到来的旅行行程、程序包交付和最近附件。 可以从一个搜索框搜索任何帐户的电子邮件、文件和联系人，或使用我们的新筛选器缩小搜索范围。
