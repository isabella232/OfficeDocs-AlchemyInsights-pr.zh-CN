# <a name="cant-select-contacts-when-composing-email"></a>撰写电子邮件时无法选择联系人

撰写新电子邮件时，一些客户在单击"到"或"抄送"按钮时看不到联系人列表。  若要选择联系人，请在"To"或 **"Cc"** 字段中开始键入，将显示相关联系人。
