# <a name="use-your-browser-to-check-spelling"></a>使用浏览器检查拼写

尽管 Outlook.com 没有拼写检查功能，但许多 Web 浏览器会自动检查拼写，包括 Microsoft Edge、Internet Explorer 10 及更高版本以及 Firefox、Chrome 和 Safari 的当前版本。 检查 Web 浏览器的选项，详细了解如何检查拼写。
