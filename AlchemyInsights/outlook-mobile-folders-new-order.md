# <a name="my-folders-are-in-a-new-order"></a>我的文件夹按新顺序排列

在 iOS 和 Android Outlook收藏夹和组后，你可能会注意到文件夹按新顺序排列。 如果你有收藏夹，你将在边栏顶部看到收藏夹部分。 可以使用靠近边栏顶部的铅笔图标添加收藏夹。 这样，你所有最重要的文件夹和组都可以在最上面轻松访问它们。
