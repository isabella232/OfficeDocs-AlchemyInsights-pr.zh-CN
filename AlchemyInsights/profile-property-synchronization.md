# <a name="profile-property-synchronization"></a>配置文件属性同步

无法将 AD 导入的其他属性从 Azure Active Directory 同步到用户配置文件应用程序。 AD 导入仅限于预先配置的属性集，以确保计时器作业的性能一致。
