# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>在 Adroid Microsoft Edge iOS 设备上阻止广告

阻止除 AdBlock Plus 认为可以接受广告之外的所有广告:
1. 选择 **…** > **设置**  > **内容阻止程序**。
2. 打开 “阻止广告”。

阻止所有广告:
1. 选择 **…** > **设置**  > **内容阻止程序**。
2. 打开 “阻止广告”。
3. 选择 **“高级设置”**。
4. 关闭 “允许可接受的广告”。
