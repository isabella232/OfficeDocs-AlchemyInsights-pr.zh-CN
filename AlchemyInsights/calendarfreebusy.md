# <a name="issues-seeing-calendar-freebusy-information"></a>查看日历忙/闲信息时的问题

如果无法在其他 Outlook 日历中查看日历忙/闲信息，或者其他人在日历中看不到忙/闲信息，请使用 Microsoft 支持和恢复助手[for Microsoft 365](https://diagnostics.office.com/)。
