# <a name="how-to-get-support-in-outlookcom"></a>如何在 Outlook.com 获取支持

抱歉，Outlook.com 不提供电话支持，但我们有专门的团队可以帮助你。
在上方的帮助框中输入你的问题或描述你的问题，然后选择获取 **帮助**。

[获取有关 Outlook.com 的帮助](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
