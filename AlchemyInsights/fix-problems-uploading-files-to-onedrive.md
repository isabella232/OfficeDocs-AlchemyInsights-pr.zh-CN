# <a name="fix-problems-uploading-files-to-onedrive"></a>修复了将文件上载到OneDrive

如果文件无法上载，请尝试以下操作：
  
- 重命名文件并重试。 不允许使用文件名中的某些文件名和字符。 
    
- 如果要同步文件，请将其从 OneDrive 文件夹 (例如，将其移动到桌面) ，当 OneDrive 同步 状态显示为"最新"时，将其移回。
