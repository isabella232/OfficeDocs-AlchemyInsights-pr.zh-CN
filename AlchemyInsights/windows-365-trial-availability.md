# <a name="windows-365-trial-availability"></a>Windows 365 试用的可用性

由于需求量很大，我们暂时暂停了 Windows 365 的新试用订阅。

我们建议你进行注册，以便在试用恢复时收到通知。 若要进行注册，请参阅 [注册以了解有关 Windows 365 的详细信息](https://aka.ms/Win365InfoNotification)。
