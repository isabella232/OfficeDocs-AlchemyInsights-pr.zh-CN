# <a name="merge-microsoft-accounts"></a>合并 Microsoft 帐户

- 无法将 Microsoft 帐户与 Office 365 工作或学校帐户合并或链接，因为两者属于不同的服务，但可以同时使用。
- 无法将 Microsoft 帐户（即使有单独的电子邮件地址）或非 Microsoft 帐户（例如 Gmail 帐户）合并或链接到现有的 Microsoft 帐户。
