# <a name="share-a-communication-site-with-external-users"></a>与外部用户共享通信网站

若要为通信网站启用外部用户共享，请按照以下步骤操作： 
  
1. 打开 **新式管理中心**- 如果你在经典管理中心，你将看到从右上方开始尝试或** 试用预览 ** 按钮。 这会将你带至新式管理中心。 
  
2. 选择 **"活动站点"。**
  
3. 选择具有通信网站 **模板的网站**。 
  
4. 从顶部 **菜单中** 选择"共享"按钮。 
  
 **注意：** 如果选择了多个网站，" **共享** "按钮将显示为已禁用。 仅选择一个网站。
