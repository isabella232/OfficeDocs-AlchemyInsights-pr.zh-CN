# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>将 Microsoft Edge 设置为 Windows 10 设备上的默认浏览器

操作步骤如下：

1. 选择“**开始**”按钮，然后输入“**默认应用**”。

1. 在搜索结果中，选择“**默认应用**”。

1. 在“**Web 浏览器**”下，选择“**Microsoft Edge**”。
