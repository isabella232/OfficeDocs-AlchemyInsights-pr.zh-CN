# <a name="public-folder-access-from-mobile-devices"></a>从移动设备访问公用文件夹

无法从移动设备访问公用文件夹。
  
如果你想要具有与公用文件夹类似的功能，并且从移动设备访问它，请参阅了解 [Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 组，了解更好的替代方法。
